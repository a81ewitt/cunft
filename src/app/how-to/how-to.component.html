How to create your NFT
Creating your NFT can be done if a few easy steps
Choose a file and add you details
  1. Choose file you wish to turn into an NFT. This does not need to be a image. You can make an NFT out of any file
  2. Your wallet address is the address that will recieve the NFT when created. This is a address you own and control
  3. Name your NFT and add any extra Attributes your want attached when the NFT is created
Submit payment
  1. Send the Total amount of ADA to the Cardano payment address. You can either scan the QR code or copy the address
  2. The Cardano Total is the sum of Cardano fee and Cunft fee. Cardano fee is the fee for using the cardano network. Cunft fee is the fee we charge to offer you this service
  3. The Received value is how much has been sent to the payment address. Please eremember it may take 1-2 minutes for the payment to update.
